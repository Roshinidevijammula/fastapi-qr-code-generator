# QR Code Generator

A full-stack QR Code Generator built using FastAPI, JavaScript, AWS S3, and Docker.

## Features

- Dynamic QR Code Generation
- REST API Architecture
- AWS S3 Integration
- Docker Containerization
- FastAPI Backend
- Interactive JavaScript Frontend

## Tech Stack

- Python
- FastAPI
- JavaScript
- AWS S3
- Docker

## Project Architecture

Frontend → FastAPI → QR Generator → AWS S3 Storage
